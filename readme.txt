パスワードをコピーするツールです
入力した丸文字を見えないようにしているのでショルダーハッキングに有効です
また、ENABLED_MINUTSをtrueに設定することで入力するパスワード+分になるのでパスワードの固定化を防止します
ただし、ここで設定するパスワード(SUCCESS_OUT_PASS)は平文なのでこのファイルを安全な場所で使用してください、
また、パスワードを設定したまま第3者に譲渡しないでください、盗み取られる可能性がございます

-- 各設定項目 --
INPUT_PASS: 画面で入力する用のパスワードを設定
ENABLED_MINUTS: INPUT_PASSの後ろに分時間の分(先頭0なし)を設定します、分単位で入力するパスが変わる仕組みです。
SUCCESS_OUT_PASS: 正しいコピーするパスワードを設定します
INVALID_OUT_PASS: 入力された値が不正の場合にコピーされる文字列を設定します
